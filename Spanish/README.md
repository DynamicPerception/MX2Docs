MX2 Spanish Documentation
=================

This repository stores the MX2 controller documentation in Spanish.
There are two versions of the same document,
The document called: "MX2 Motion Controller En.pdf" maintains 
the original menus in English, 
while in the document: "MX2 Motion Controller EJD.pdf" also have been translated menu entries.

Este repositorio almacena la documentación del controlador MX2 en Español.
Existen dos versiones del mismo documento. El documento llamado: "MX2 Motion Controller En.pdf"
mantiene los menús originales en Inglés, 
mientras que en el documento: "MX2 Motion Controller EJD.pdf" se han traducido tambien las entradas de menú.
